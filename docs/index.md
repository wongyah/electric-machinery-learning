# 轻松入门电机学

轻松入门电机学是采用经典的技术写作理论（主题化写作）重新编写的一本经典工科教材，不仅适合**零基础的读者**用来**快速入门**电机学，也非常适合**专业人士**作为手边教材**随时查阅**。

本教材精心提取了电机学中的核心概念和理论，以图文并茂的形式娓娓道来。对于抽象的理论过程，还特意制作了栩栩如生的动画。每个章节独立成篇，即开即读，不需要从第一章第一节开始顺序阅读。
